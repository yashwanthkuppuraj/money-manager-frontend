# Money Manager Frontend

A modern, responsive personal finance dashboard built to help you track income, expenses, and budget effectively.

## Project Overview

Money Manager offers a clean, intuitive interface for managing your personal finances. With features like comprehensive analytics, secure account management, and a customizable budget planner, it puts you in control of your financial health.

## Key Features

- **Dashboard**: Real-time overview of your finances with Income/Expense summaries.
- **Analytics**: Visual breakdown of spending habits by category and time period (Weekly/Monthly/Yearly).
- **Secure Authentication**: User accounts protected by JWT authentication.
- **Transaction Management**: easy-to-use forms for adding incomes, expenses, and transfers.
- **Budget Planner**: Set and track limits for different categories (Planned).
- **Responsive Design**: Works seamlessly on Desktop and Mobile.
- **Safe & Private**: Your data is isolated and protected.

## Tech Stack

- **Framework**: ReactJS (Vite)
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **HTTP Client**: Axios
- **Routing**: React Router DOM
- **Date Handling**: Date-fns

## Setup Instructions

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Configuration**:
    Ensure the backend API is running (usually on port 5000). The `vite.config.js` is set up to proxy API requests.

3.  **Run Application**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) in your browser.
